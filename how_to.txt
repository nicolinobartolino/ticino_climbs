Drop .gpx file/s in the gpx folder, name them what you want them to show as in the route's start tooltip.
Run: python3 auto_add_planned_gpx.py

This populates the gpx_files.json with the gpx name, adds route to the map and to the climbs_info.json with a status of planned 

2nd python script to extract and write Distance, Elevation and Average gradient
Run: python3 auto_update_climb_info.py

Cheeky extra script to populate start names from the gpx file name:
Run: python3 update_start_names.py

Script to update progress bar
Run: python3 update_climb_stats.py
(Re-run if changing a climb from planned > ridden)

Script to add Category to climb start tooltip
python3 auto_assign_climb_categories.py

Changing status from planned to ridden changes the route to red (manual)
PB and Strava link can be edited in climbs_info.json manually
